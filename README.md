# Image Encryption Tool

A simple Python project that encrypts and decrypts images using pixel manipulation techniques.

## Features
- Encrypt images using XOR operation
- Decrypt encrypted images using the same key
- Pixel swapping for image transformation
- User-defined encryption key
- Easy-to-use console interface

## Technologies Used
- Python
- Pillow (PIL)

## How to Run
1. Install the Pillow library.
2. Run the Python program.
3. Enter the image path.
4. Enter the encryption key.
5. Choose Encrypt or Decrypt.

## Learning Outcomes
- Image processing with Pillow
- Pixel manipulation techniques
- Basic image encryption concepts
- File handling in Python
